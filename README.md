# Bill-Splitter
A simple Python program to split a bill between multiple participants.

---

## Features

- Split bill evenly or by custom percentages
- Input validation for numbers and names
- Prevent duplicate names
- Automatically assigns remaining percentage to the last person
- Clear summary of each person's share

##Example
 Welcome to expense splitter!
Enter the total bill amount: $100
Enter the name of participant: ahmed
Enter the name of participant: ali
Enter the name of participant: done

Do you want to split the bill evenly? (yes/no): no

Ahmed will take the remaining 50%.

--- split summary ---
Ahmed     : $50.00
Ali       : $50.00
